# balanceamento-de-delimitadores
Aplicação que verifica delimitadores em um arquivo de código por meio de análise de pilha; 
Linguagem utilizada: C++;
Atividade desenvolvida para a disciplina de Algoritmos e Estruturas de Dados 1 - 2025.

Arquivos
- pilha.cpp e pilha.hpp: contém a implementação e a definição, respectivamente, das classes pilha e nó, que são necessárias para a estrutura de dados base do projeto;
- balanceamento.cpp: contém as funções para leitura de arquivo e análise dos delimitadores;
- main.cpp: pede ao usuário o arquivo a ser analisado e chama a função para iniciar a análise.
